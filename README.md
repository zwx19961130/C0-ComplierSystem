# 编译原理第三次实验

## 实验目的：

分析所给的PL/0文法，输入符合PL/0文法的源程序，输出pCode

## 实验收获：
1.	了解了什么是编译程序。
2.	了解了编译程序的分类与表示
3.	了解了编译程序的组成结构和工作过程
4.	了解了典型编译程序的逻辑阶段划分
5.	了解了编译程序的构造要素
6.	了解了编译程序的实现方式


## 实验步骤：

1.通过readFile()读入源代码
2.通过GETSYM()函数读入的一个单词进行分类，确定单词属于哪一类，
3.输出pCode 的结果。

 ## 实验感想：

通过这次实验，我知道了怎样编写PL0编译系统，了解了编写编译系统的大致步骤，对编译过程有了更深的理解。
